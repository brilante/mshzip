MathJax 렌더링 테스트

테스트 1: Display Math (블록 수식)

원주율 공식 (라이프니츠 급수):

\[\pi = 4 \sum_{n=0}^{\infty} \frac{(-1)^n}{2n+1}\]

테스트 2: Inline Math (인라인 수식)

아인슈타인의 유명한 공식: \(E = mc^2\) 는 에너지와 질량의 관계를 나타냅니다.

테스트 3: 이차방정식 근의 공식

\[x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]

테스트 4: 오일러 공식

수학에서 가장 아름다운 공식:

\[e^{i\pi} + 1 = 0\]

테스트 5: 바젤 문제

\[\frac{\pi^2}{6} = \sum_{n=1}^{\infty} \frac{1}{n^2}\]

테스트 6: 복합 텍스트 (Bold + LaTeX)

**굵은 텍스트**와 수식 \(\pi\)가 함께 있는 경우입니다.

또 다른 예: **중요!** 원의 둘레는 \(C = 2\pi r\) 입니다.
