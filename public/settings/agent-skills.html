Claude Code 사용 예시

1. 기획 문서 읽기

마인드맵 노드의 기획 문서를 조회합니다.

/기획 R {노드ID}

# 예시
/기획 R KRI216

2. 기획 문서 쓰기

작업 결과를 노드의 기획 문서에 기록합니다. 본문 기록과 첨부 파일 생성 두 가지 모드를 지원합니다.

# 본문에 내용 추가 (HTML 모드)
/기획 W {노드ID}

# 예시: 현재 작업 내용을 본문에 기록
/기획 W KRI216

# 첨부 파일로 저장 (마크다운 파일 모드)
/기획 W {노드ID} '{파일명}'

# 예시: 분석 결과를 별도 파일로 첨부
/기획 W KRI216 '분석결과'

본문 모드: 노드의 HTML 본문에 내용을 추가합니다.
파일 모드: 노드 하위에 .md 첨부 파일을 생성합니다. 같은 노드에 여러 파일을 첨부할 수 있습니다.

3. 노드 ID 확인

마인드맵에서 노드를 선택하면 노드 정보에 ID가 표시됩니다.

# 노드 ID 형식 예시
KRI216, ABC001, 5KXG2GYFEH
