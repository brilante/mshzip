# 환불 정책

**시행일: 2025년 1월 1일**
**최종 수정일: 2025년 1월 19일**

MyMind3(이하 "회사")는 고객님의 만족을 최우선으로 생각하며, 투명하고 공정한 환불 정책을 운영합니다.

---

## 제1조 (적용 범위)

이 환불 정책은 MyMind3 서비스의 모든 유료 구독 상품에 적용됩니다.

### 적용 대상 상품
- Lite 요금제
- Standard 요금제
- Pro 요금제
- Max 요금제

---

## 제2조 (결제 수단)

### 1. 사용 가능한 결제 수단

| 결제 수단 | 지원 여부 | 비고 |
|-----------|----------|------|
| 신용카드 | O 지원 | Visa, Mastercard, American Express, JCB |
| 체크카드 | O 지원 | 신용카드 네트워크 연동 카드에 한함 |
| 페이팔 | X 미지원 | - |
| 계좌이체 | X 미지원 | - |
| 가상화폐 | X 미지원 | - |

### 2. 결제 처리
- 모든 결제는 Stripe를 통해 안전하게 처리됩니다.
- 결제 정보는 PCI DSS 준수 환경에서 보호됩니다.

## 제3조 (환불 정책)

### 1. 즉시 환불 원칙

**MyMind3는 "즉시 환불" 정책을 시행합니다.**

| 항목 | 내용 |
|------|------|
| 환불 요청 | 고객이 원하는 즉시 가능 |
| 환불 처리 시간 | 요청 후 즉시 처리 |
| 환불 금액 | 남은 구독 기간에 대한 일할 계산 환불 |
| 환불 방법 | 결제한 신용카드로 자동 환불 |

### 2. 환불 가능 기간

- **구독 기간 내 언제든지** 환불 요청이 가능합니다.
- 별도의 환불 기한 제한이 없습니다.

### 3. 환불 금액 계산

```
환불 금액 = 결제 금액 × (남은 일수 / 전체 구독 일수)
```

**예시:**
- 월 구독(30일) $12.10 결제
- 15일 사용 후 환불 요청
- 환불 금액: $12.10 × (15/30) = **$6.05**

---

## 제4조 (환불 절차)

### 1. 환불 요청 방법

| 방법 | 절차 |
|------|------|
| 서비스 내 설정 | 설정 > 구독 관리 > 환불 요청 |
| 이메일 | support@mymind3.com으로 요청 |
| 고객센터 | 웹사이트 내 1:1 문의 |

### 2. 환불 처리 단계

1. **환불 요청 접수** - 고객이 환불 요청 제출
2. **즉시 처리** - 시스템에서 자동으로 환불 처리
3. **구독 해지** - 해당 구독 즉시 해지
4. **환불 완료** - 카드사로 환불 요청 전송

### 3. 환불 소요 시간

| 단계 | 소요 시간 |
|------|----------|
| 환불 처리 | 즉시 (자동 처리) |
| 카드사 반영 | 3-5 영업일 |
| 명세서 반영 | 다음 결제 주기 |

※ 카드사별로 환불 반영 시간이 상이할 수 있습니다.

---

## 제5조 (환불 후 서비스 이용)

### 1. 서비스 접근

- 환불 처리 즉시 유료 서비스 접근이 종료됩니다.
- 무료(Free) 요금제로 자동 전환됩니다.

### 2. 데이터 보존

| 항목 | 처리 |
|------|------|
| 마인드맵 데이터 | 유지 (무료 요금제 한도 내) |
| 계정 정보 | 유지 |
| 결제 이력 | 영구 보존 (법적 요구사항) |

### 3. 재구독

- 환불 후에도 언제든지 다시 구독할 수 있습니다.
- 이전 데이터는 재구독 시 복원됩니다.

---

## 제6조 (환불 불가 사항)

다음의 경우에는 환불이 제한될 수 있습니다:

### 1. 환불 제한 사항

- **크레딧 소진**: 이미 사용한 크레딧에 대해서는 환불되지 않습니다.
- **프로모션 혜택**: 무료 체험 기간 또는 무료 크레딧은 환불 대상이 아닙니다.
- **부정 사용**: 서비스 약관 위반으로 인한 계정 정지의 경우

### 2. 부분 환불 적용

| 상황 | 환불 |
|------|------|
| 일부 크레딧 사용 | 미사용 기간에 대한 일할 환불 |
| 보너스 크레딧 사용 | 보너스 제외, 결제 금액 기준 환불 |

---

## 제7조 (자동 결제 취소)

### 1. 자동 갱신 취소

- 구독은 자동으로 갱신됩니다.
- 갱신을 원하지 않으시면 다음 결제일 전에 취소해주세요.

### 2. 취소 방법

1. **설정** > **구독 관리** 접속
2. **자동 갱신 취소** 클릭
3. 취소 확인

### 3. 취소 후 서비스

- 취소 후에도 현재 결제 기간 종료까지 서비스 이용 가능
- 기간 종료 후 무료 요금제로 자동 전환
