무료 SSL/TLS 인증서를 발급받는 방법을 알아보세요

Let's Encrypt

가장 인기 있는 무료 SSL 인증서 발급 기관

완전 무료 자동 갱신 90일 유효 와일드카드 지원

설치 방법 (Certbot)

1

Certbot 설치

# Ubuntu/Debian sudo apt update sudo apt install certbot python3-certbot-nginx # CentOS/RHEL sudo yum install certbot python3-certbot-nginx
2

인증서 발급

# Nginx sudo certbot --nginx -d example.com -d www.example.com # Apache sudo certbot --apache -d example.com -d www.example.com # 수동 발급 (Standalone) sudo certbot certonly --standalone -d example.com
3

자동 갱신 설정

# 갱신 테스트 sudo certbot renew --dry-run # Cron으로 자동 갱신 (이미 설정되어 있을 수 있음) 0 0 * * * /usr/bin/certbot renew --quiet

Cloudflare SSL

CDN과 함께 제공되는 무료 SSL

완전 무료 자동 갱신 설정 간편 CDN 포함

설정 방법

1

Cloudflare 가입

cloudflare.com에서 무료 계정을 생성하세요.

2

도메인 추가

대시보드에서 "Add Site"를 클릭하고 도메인을 입력합니다.

3

네임서버 변경

도메인 등록 업체에서 네임서버를 Cloudflare가 제공하는 NS로 변경합니다.

4

SSL 모드 설정

SSL/TLS 설정에서 "Full" 또는 "Full (strict)" 모드를 선택하세요.

  • Flexible: 브라우저 ↔ Cloudflare만 암호화
  • Full: 전체 암호화 (서버 인증서 검증 안 함)
  • Full (strict): 전체 암호화 (서버 인증서 검증)

Self-Signed 인증서

개발/테스트 환경용 자체 서명 인증서

즉시 생성 인터넷 불필요 브라우저 경고 개발용만

생성 방법 (OpenSSL)

1

개인키 생성

openssl genrsa -out server.key 2048
2

CSR 생성

openssl req -new -key server.key -out server.csr \ -subj "/C=KR/ST=Seoul/L=Seoul/O=MyCompany/CN=localhost"
3

자체 서명 인증서 생성

openssl x509 -req -days 365 -in server.csr \ -signkey server.key -out server.crt
4

한 줄로 생성 (간편 방법)

openssl req -x509 -nodes -days 365 -newkey rsa:2048 \ -keyout server.key -out server.crt \ -subj "/C=KR/ST=Seoul/L=Seoul/O=Dev/CN=localhost"
