해시 함수란?

해시 함수는 임의의 데이터를 고정된 길이의 값으로 변환하는 함수입니다.

  • MD5: 128비트 (32자) - 빠르지만 보안용으로 권장하지 않음
  • SHA-1: 160비트 (40자) - 더 이상 보안용으로 권장하지 않음
  • SHA-256: 256비트 (64자) - 현재 널리 사용됨
  • SHA-512: 512비트 (128자) - 가장 안전
