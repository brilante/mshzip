# Ralph Loop 자동 반복 실행 스킬 (Phase3→CC)

## 트리거
`랄프로`, `랄프 루프`, `ralph`, `/랄프`

## 동작 원리

1. **목표 분석**: 사용자 오더에서 목표와 탈출조건 추출
2. **탈출조건 정의**: 정량적 기준 + max_turns(20) 설정
3. **반복 실행**:
   - 구현 (코드 수정)
   - 검증 (테스트 실행)
   - 탈출조건 체크
   - 미충족 → 재수정
   - 충족 → 자동 종료
4. **결과 보고**: 총 반복 횟수, 최종 결과

## 탈출조건 유형

| 유형 | 예시 | 판단 |
|------|------|------|
| 정량적 | 응답 < 3초 | 자동 측정 |
| 기능적 | 로그인 성공 | 브라우저/API 테스트 |
| 품질적 | 에러 0건 | lint/test 실행 |

## 3중 안전장치

1. **탈출조건** (논리적 종료)
2. **max_turns: 20** (물리적 종료)
3. **금지 동작**: `rm -rf`, `DROP TABLE`, `npm publish`, `git push`

## 사용 예시
```
/랄프 "API 응답 시간 3초 이내로 최적화"
/랄프 "로그인 기능 구현 후 브라우저 테스트 통과"
```
