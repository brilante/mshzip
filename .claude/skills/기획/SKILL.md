# 기획 문서 스킬 (Phase1→CC: 기획 전문가)

## 역할
기획 문서 읽기/쓰기 전문가. 마인드맵 노드를 통해 기획 문서를 관리.

## 명령어

```bash
/기획 R <노드ID>           # 노드 읽기
/기획 W <노드ID>           # 노드에 내용 추가 (append)
/기획 W <노드ID> '파일명'  # 첨부 파일 생성
/기획 LOG <노드ID>         # 명령 이력 기록 (전용 모드)
```

### LOG 명령 상세

**용도**: 사용자 명령 수행 결과를 이력 노드에 구조화하여 기록하는 전용 모드.

**기록 형식**:
```
#### [시간] 명령 제목
- **명령**: 사용자 명령 원문
- **액션**: 수행한 도구명 + 대상 파일 목록
- **결과**: 성공/실패 + 사유
- **관계**: 이전 명령과의 연속/변경/충돌
```

**동작**:
1. 위 형식으로 기록 내용을 자동 구성
2. 대상 노드에 append 모드로 추가 (`/기획 W` 내부 호출)
3. 기록 후 반드시 읽기(R)로 검증

**기본 이력 노드**: `CFLA6IJAND` (CLAUDE.md 참조)

**사용 예시**:
```bash
/기획 LOG CFLA6IJAND        # 이력 노드에 현재 명령 결과 기록
```

## 규칙
- 기본 동작은 **추가(append)**
- "업데이트" 명시 시에만 덮어쓰기
- 노드 생성 후 반드시 읽기(R)로 검증
- LOG 모드는 항상 append + 타임스탬프 포함

## 노드ID 패턴
영숫자 10자리 (예: `YU23DJ8GJQ`) → 자동 인식 → 이 스킬 사용
